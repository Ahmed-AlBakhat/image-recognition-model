# Image Recognition Model

## Overview
This project uses Google's Teachable Machine and TensorFlow to classify images into two categories:

- Phone
- Watch

## Tools Used
- Google Teachable Machine
- Python 3.13
- TensorFlow
- tf_keras
- Pillow
- NumPy

## Project Files
- predict.py
- keras_model.h5
- labels.txt
- test.jbg.jpg
- test2.jpg

## Results
The model successfully classified:

- Phone image with high confidence.
- Watch image with high confidence.
